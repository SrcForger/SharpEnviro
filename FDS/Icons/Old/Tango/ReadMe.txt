Tango Icons for the SharpEnvironment
www.sharpe-shell.org

How to install: copy this Folder into the /Icons Folder of SharpE.
After this, activate the Icons in the SharpCenter (Themes -> Edit -> Icon List).

The icons in this repository are herefore released into the Public Domain.

http://tango.freedesktop.org
Tango Version: 0.8.90

The Icons "icon.drive.networkoff.ico", "icon.drive.networkon", "icon.folder.pictures.ico", "icon.folder.movies.ico"
and "icon.folder.sound.ico" are based on Tango Icons and made by me.

The IconSet.xml is Beerware.
