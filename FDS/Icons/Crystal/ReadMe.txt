Crystal Icons for SharpEnvironment Final Version

How to install: copy this Folder into the /Icons Folder of SharpE.
After this, activate the Icons in the SharpCenter (Themes -> Edit -> Icon List).

icon.drive.floppy
icon.drive.cdrom
icon.file.config
Suggestions for replacement of these Icons? Email me at jakob.gillich <at> googlemail.com

If you want the KDE Icon for the Menu and the Computer button, rename the kmenu.ico to icon.mycomputer.ico
